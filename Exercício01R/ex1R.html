Meu aprendizado em PW1

    Meu ambiente

  • VsCode

    • Através da interface de desenvolvimento Vscode eu posso digitar meus códigos.

  • Github e Git Bash

    • No Github nós guardarmos nossos projetos feitos por diversos tipos de linguagens de desenvolvimento, utlizamos o terminal Gitbash para enviar tais projetos.

    O Gitbash não é o único terminal disponível, existem vários outros! No terminal no Linux por exemplo a funcionalidade do Git é nativa.

    Linguagens Utilizadas

  • HTML

    • HTML é uma Linguagem de Marcação de Hipertexto, usamos o HTML para publicarmos algum conteudo na web.

    Abaixo veremos alguns exemplos de tags em HTML:

    1. p é uma tag de parágrafo.

    2. h1 ao h6 é uma tag de Título, sendo o menor h6, e o maior h1.

    3. br é uma tag de quebra de linha.

  • CSS

    • O CSS é uma linguagem de estilização, usamos ela para mudar a cor de fundo da página, mudar cor das letras, etc.

    Temos como exemplo, a estilização style, que serve para mudar as cores de títulos, parágrafos e entre outros.
